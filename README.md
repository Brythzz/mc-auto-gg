# Auto GG
Minecraft 1.8.9 Auto "gg" mod for the [Hypixel](https://hypixel.net) server (mc.hypixel.net)

### Supported languages
- English
- French

### Usage
`/autogg [delay]` - Enable/disable the mod (optional delay in ms)

### Default settings
Enabled: `false`
<br/>
Delay: `1000ms`
